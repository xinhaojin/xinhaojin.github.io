DBeaver连接MySQL和IoTDB数据库步骤

一、DBeaver 连接 MySQL 数据库步骤

  1. 新建数据库连接

    • 打开 DBeaver，选择新建数据库连接。
    • 在数据库类型中选择 MySQL。

  2. 编辑驱动设置

    • 进入驱动设置页面，选择库选项。

  3. 添加驱动文件

    • 添加 MySQL 驱动文件，例如 mysql-connector-java-8.0.17.jar。
    • https://gitee.com/liwei125/dbeaver-driver-all/blob/master/drivers/mysql/mysql8/mysql-connector-java-8.0.17.jar

  4. 设置驱动类

    • 找到驱动类为 com.mysql.cj.jdbc.Driver，点击确定完成设置。

二、DBeaver 连接 IoTDB 数据库步骤

（一）下载 IoTDB JDBC 驱动

  • 从IoTDB官网https://iotdb.apache.org/zh/UserGuide/latest/Ecosystem-Integration/DBeaver.html下载与您 IoTDB 版本适配的 JDBC 驱动。

（二）安装 JDBC 驱动

  1. 打开 DBeaver，点击菜单栏中的 “数据库”。
  2. 选择 “驱动管理器”，点击 “新建”。
  3. 在弹出窗口中，填写以下信息：
    • 类名：org.apache.iotdb.jdbc.IoTDBDriver
    • URL 模板：jdbc:iotdb://127.0.0.1:6667/
    • 默认端口：6667
    • 默认用户：root
  4. 点击 “添加文件”，选择之前下载的 JDBC 驱动 jar 文件，然后点击 “确定”。

（三）连接 IoTDB 数据库

  1. 在 DBeaver 中，点击 “新建连接”，搜索 “IoTDB”。
  2. 输入数据库的连接信息，包含 IP 地址、端口号、用户名和密码。
  3. 点击 “测试连接”，确保连接成功。
  4. 若测试通过，点击 “确定” 完成连接设置。

通过上述步骤，即可在 DBeaver 中成功连接 MySQL 和 IoTDB 数据库，进而执行 SQL 查询和管理数据库。
