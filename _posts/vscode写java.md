#### 安装插件

扩展程序搜java就行，主要是安装language support

![](https://cdn.staticaly.com/gh/xinhaojin/imgs-host@master/past/2021/04/image-38.png)

安装好后在设置里搜索java.home

![](https://cdn.staticaly.com/gh/xinhaojin/imgs-host@master/past/2021/04/image-39.png)

设置里搜索java runtime，选择已安装的jdk，然后reload window

![](https://cdn.staticaly.com/gh/xinhaojin/imgs-host@master/past/2021/04/image-50-1024x427.png)

#### 测试

ctrl+shift+p输入java，选择创建新项目

![](https://cdn.staticaly.com/gh/xinhaojin/imgs-host@master/past/2021/04/image-42.png)

选择no build tools

![](https://cdn.staticaly.com/gh/xinhaojin/imgs-host@master/past/2021/04/image-43.png)

选择文件夹

![](https://cdn.staticaly.com/gh/xinhaojin/imgs-host@master/past/2021/04/image-44.png)

输入项目名

![](https://cdn.staticaly.com/gh/xinhaojin/imgs-host@master/past/2021/04/image-45.png)

生成了项目文件，选择app.java

![](https://cdn.staticaly.com/gh/xinhaojin/imgs-host@master/past/2021/04/image-51.png)

helloworld输出成功
